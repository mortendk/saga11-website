## Quick start

0. get 11ty

```
$ npm install -g @11ty/eleventy
```

1. install saga

```
$ npm install
```

2. spin Saga11 to build the site

```
$ npm  start
```

http://localhost:8080

3. build the production site

```
npm run build
```
